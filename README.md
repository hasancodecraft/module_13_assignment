# Module 13 Assignment

This is a Flutter assignment project based on counter, navigation, bottom navigation and tab view.

## Features

* Home screen
* Details screen
* Counter increment using setState
* Pass counter value to details screen
* Navigate to details screen using Navigator.push
* Return back using Navigator.pop
* Bottom navigation with Home and About
* TabBar and TabBarView used in home section

## Used Widgets and Concepts

* StatefulWidget
* initState
* dispose
* setState
* Scaffold
* AppBar
* BottomNavigationBar
* TabBar
* TabBarView
* Navigator
